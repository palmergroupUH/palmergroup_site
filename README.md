Palmer Research Group Website
=============================

This is the official website for the Palmer Research Group at the University of Houston. It showcases our research in condensed matter theory, molecular simulation, and materials design.

Project Structure
-----------------
- index.html: Main page structure.
- style.css: Custom styling.
- script.js: Logic for interactive elements.
- images/: Folder for all image assets.
- resources/: All PDF resources.

How to Run Locally
------------------
1. Clone the repository:
   git clone https://github.com//palmergroupUH/palmergroup_site.git

2. Open index.html in your web browser.
